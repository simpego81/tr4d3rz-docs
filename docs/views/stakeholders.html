Una popolazione di agenti evolutivi.

Ogni agente porta un genoma unico.

Il genoma è un L-System —

regole che si auto-applicano per generare strutture.

F → FF+[+F-F-F]-[-F+F+F]

Le regole generano un albero decisionale 3D.

Ogni ramo è un percorso di valutazione.

I dati di mercato attraversano l'albero.

Ogni nodo prende una decisione: compra, vendi, o attendi.
